# Robot Arm

## Overview
This repository contains a robotic arm control system written in Python. The project aims to provide an intuitive interface for controlling a robotic arm with various functionalities.

## Features
- Control robotic arm movement.
- Support for various motion types including linear and rotational movements.
- User-friendly command interface.
- Modular design for easy extension and integration.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Frochoso/robot-arm.git
   ```
2. Navigate to the project directory:
   ```bash
   cd robot-arm
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To execute the program, run:
```bash
python main.py
```
